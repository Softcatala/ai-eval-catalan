Transcripció local amb Whisper: eines amb interfície gràfica

Whisper és el model de reconeixement de veu d'OpenAI, disponible de forma gratuïta i que s'executa completament en local, sense enviar dades a cap servidor extern. Si prefereixes evitar la línia de comandes, aquí tens les principals opcions amb interfície gràfica.

Whisper Desktop

Aplicació per a Windows i macOS basada en whisper.cpp. Permet carregar un fitxer d'àudio o vídeo i obtenir la transcripció amb uns quants clics, sense necessitat d'instal·lar Python ni cap dependència addicional.

  • Interfície senzilla i minimalista
  • Permet triar el model i l'idioma
  • Exporta la transcripció en text pla o SRT
  • Funciona bé a la CPU, sense GPU

Subtitle Edit

Editor de subtítols molt complet per a Windows (funciona també a Linux via Wine o .NET). Té integració directa amb Whisper per generar subtítols automàticament a partir d'àudio o vídeo.

  • Ideal si el resultat final ha de ser un fitxer de subtítols (SRT, VTT, ASS...)
  • Permet editar i sincronitzar subtítols manualment
  • Suporta diversos models de Whisper
  • Molt útil per a contingut de vídeo

Transcribro

Aplicació de codi obert per a Linux amb interfície gràfica GTK. Pensada específicament per a l'escriptori GNOME, permet transcriure àudio de forma còmoda sense obrir cap terminal.

  • Integrada a l'escriptori Linux
  • Suporta múltiples idiomes, inclòs el català
  • Exporta el resultat en text pla

Whisper al navegador (sense instal·lació)

Existeixen implementacions de Whisper que s'executen directament al navegador web, sense instal·lar res. Utilitzen WebAssembly o WebGPU per córrer el model localment al teu ordinador, de manera que l'àudio no surt mai del dispositiu.

  • Sense instal·lació ni configuració
  • Privacitat garantida: tot es processa al teu dispositiu
  • Velocitat limitada comparada amb aplicacions natives
  • Recomanat per a transcripcions ocasionals

Opció per a usuaris avançats: faster-whisper

Si et mous còmodament amb Python i vols més control, faster-whisper és la millor opció per línia de comandes. És una reimplementació optimitzada de Whisper que pot ser fins a quatre vegades més ràpida que l'original, amb un consum de memòria molt inferior. Compatible amb CPU i GPU, i ideal per integrar en scripts o automatitzacions.
